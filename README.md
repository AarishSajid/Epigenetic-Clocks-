# Epigenetics & Aging Analysis

Bioinformatics analysis project exploring epigenetics and aging patterns using computational workflows.

## Project Structure
- `notebooks/` - Jupyter notebooks
- `data/` - raw and processed datasets
- `outputs/` - generated figures/results

## Setup

```bash
pip install -r requirements.txt
jupyter notebook
```

## Run
Open `notebooks/original_analysis.ipynb` and run cells sequentially.
